Conceitos Básicos de Python

Estruturas de Seleção

Conceito: Permitem que o programa tome decisões com base em condições.

Aplicação: Usadas para executar blocos de código diferentes dependendo de condições avaliadas.

idade = int(input("Digite sua idade: "))
if idade < 18:
    print("Você é menor de idade.")
elif idade < 65:
    print("Você é adulto.")
else:
    print("Você é idoso.")

Estruturas de Repetição

Conceito: Permitem que um bloco de código seja executado várias vezes.

Aplicação: Usadas para iterar sobre sequências, repetir ações enquanto uma condição for verdadeira, ou até certo número de vezes.

for i in range(5):
    print(f"Contagem: {i + 1}")

contador = 0
while contador < 5:
    print(f"Contagem: {contador + 1}")
    contador += 1

Vetores e Matrizes

Conceito: Estruturas de dados que armazenam coleções de elementos em uma sequência ou em múltiplas dimensões.

Aplicação: Utilizadas para armazenar diversos tipos de dados organizados, como listas (vetores) e listas de listas (matrizes).

# Vetor (lista)
vetor = [1, 2, 3, 4, 5]
print("Vetor:", vetor)

# Matriz (lista de listas)
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print("Matriz:")
for linha in matriz:
    print(linha)

Funções e Procedimentos

Conceito: Blocos reutilizáveis de código que realizam tarefas específicas.

Aplicação: Usadas para organizar código, evitar repetição e facilitar manutenção.

def soma(a, b):
    return a + b

resultado = soma(5, 3)
print("Resultado da soma:", resultado)

Tratamento de Exceções

Conceito: Mecanismo para lidar com erros que podem ocorrer durante a execução do programa.

Aplicação: Usado para evitar que o programa seja interrompido e para fornecer respostas adequadas a erros.

try:
    numero = int(input("Digite um número: "))
    print("Você digitou:", numero)
except ValueError:
    print("Erro: valor digitado não é um número válido.")
